## Installation

- `git clone git@github.com:wwilsman/new-tab-plus`
- Add Unsplash API credentials to `.env`
- `yarn && yarn build`

### Chrome

- Navigate to `chrome://extensions`
- Click "Load unpacked extension..."
- Select `new-tab-plus/build`
